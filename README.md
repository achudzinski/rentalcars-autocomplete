<h2>Install</h2>

```bash
npm install
```

<h2>Usage</h2>

```bash
npm run webpack:watch
```

Application is available on address <a href="http://localhost:8081/app/">http://localhost:8081/app/</a>.

<h2>Tests</h2>

To run end-to-end tests run:

```bash
npm run e2e-test
```
